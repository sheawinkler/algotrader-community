# Community Edition Contract

This repository is intentionally limited.

## Included

- Rust CLI for local backtest/demo flows
- simple entry framework based on:
  - price momentum
  - volume surge
  - flow imbalance
- deterministic sample dataset
- local reporting in SOL terms

## Excluded

- private provider wiring
- low-latency streaming infrastructure
- Yellowstone gRPC
- Bitquery/CoreCast
- sidecar model serving
- wallet-following and cohort intelligence
- live execution
- deploy/relaunch automation
- proprietary predictive scoring and calibration paths
